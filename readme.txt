#This program is implemented in python3. The python file is: knaivebayesclassifier.py

#To run the classifier, just run naivebayesclassifier.py with 2 argument [name of training set] [name of test set]

Copy the below command line for convinience:

python3 naivebayesclassifier.py breast-cancer-training.csv breast-cancer-test.csv


------------------------------------------
What does this program do
It use the training data to build a naive bayes classifier, then predict the class for each test instance

----------------
Note: data file must be in the 'data' folder under the same directory with knaivebayesclassifier.py
